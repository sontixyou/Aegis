# Aegis

A TypeScript web server built with Hono framework.

## Getting Started

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

The server will start at `http://localhost:3000`

### Build

```bash
npm run build
npm start
```

### Code Quality

```bash
# Lint code
npm run lint

# Auto-fix linting issues
npm run lint:fix

# Format code
npm run format
```

### Commit

This project uses Commitizen for consistent commit messages:

```bash
npm run commit
```

## Tech Stack

- **Runtime**: Node.js
- **Framework**: Hono
- **Language**: TypeScript
- **Linter/Formatter**: Biome
- **Build Tool**: TSC
